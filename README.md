# practices
.bin格式, np.tofile() 和 np.fromfile()    ---list
.npy格式，np.save() 和 np.load()   ----list 
.txt 或者 .csv格式，np.savetxt() 和 np.loadtxt()  -----str
.h5 格式，h5py.File(，’r’ 或者 ‘w’)    ---dict  
.pkl 格式， pickle.dump()和pickle.load()  ---csv

每一次实践都是一次纠错的过程，希望代码能写的越来越简练，bug越来越少。
